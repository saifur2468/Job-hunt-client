# 💼 Job Hunt Web Application

A modern **Job Hunt Platform** where users can explore jobs, apply, and manage applications easily. This project is built using **React, Tailwind CSS, SweetAlert, Node.js, and Express.js**.

# Live LinK : https://job-hunting-ce638.web.app/

## 🚀 Features

### 👤 User Features

* 🔍 Browse available jobs
* 📄 View job details
* 📨 Apply for jobs
* 📋 Manage applied jobs
* ❌ Remove applications
* 🧑‍💼 Admin Dashboard
* 📊 Analytics

### ⚙️ System Features

* ⚡ Fast and responsive UI
* 🔔 Beautiful alerts using SweetAlert
* 🌐 REST API integration
* 🔄 Dynamic data loading
## 🧑‍💻 Tech Stack

### 🎨 Frontend

* React.js
* Tailwind CSS
* SweetAlert2
* Axios / Fetch API

### ⚙️ Backend

* Node.js
* Express.js

### 🗄️ Database

* MongoDB
